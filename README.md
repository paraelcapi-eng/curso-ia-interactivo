# curso-ia-interactivo
curso IA
ID,Nombre del Reto,Objetivo Pedagógico,Instrucción para el Alumno
01,El Semáforo Humano,Clasificación básica,"Entrena 3 clases: ""Rojo"" (mano abierta), ""Amarillo"" (puño), ""Verde"" (dedo arriba)."
02,Detector de Emociones,Reconocimiento de patrones,"Entrena ""Feliz"" vs ""Enojado"". Prueba si funciona si te tapas la boca."
03,Inspector de Frutas,Variabilidad de datos,Usa una manzana real y una dibujada. ¿La IA nota la diferencia?
04,El Lector de Mentes,Sesgo de datos,Entrena con tu mano derecha. Prueba si reconoce la izquierda.
05,Eco-Clasificador,Impacto ambiental,Diferencia entre una botella de plástico y una de vidrio.
06,Estatua vs Humano,Movimiento,"Entrena ""Persona quieta"" vs ""Persona moviéndose""."
07,Criptografía Visual,Abstracción,"Entrena la IA para reconocer ""A"" cuando muestres un lápiz y ""B"" con un borrador."
08,El Árbitro de Piedra,Juego Lógico,"Entrena ""Piedra"", ""Papel"" o ""Tijera""."
09,Corrector de Postura,Salud Digital,"Entrena ""Bien sentado"" vs ""Encorvado""."
10,Detector de Marcas,Marketing y Visión,Diferencia entre el logo de Google y el de Apple (puedes usarlos en el celular).
